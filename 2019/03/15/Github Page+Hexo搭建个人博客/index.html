将项目源文件推送到Github上，方便迁移

  1. 将GitHub上的项目克隆到本地

  2. 删除除了.git外的所有文件

  3. 将Hexo创建的项目中的所有文件复制进来，安装的依赖文件和使用Hexo命令生成的静态文件可以不用复制

  4. 创建分支Hexo

  5. 将复制过来的文件提交到暂存区

  6. 推送到GitHub

  7. 后面在其它电脑使用的时候，直接克隆分支hexo项目，再安装依赖
